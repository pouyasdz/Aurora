## Getting Started

just clone repository and install pakages then `npm start`

### Prerequisites

install pakages:

- NPM
`npm install`

- YARN
`yarn install`

### Installation

add some .env file and add Your Discord Bot Token 

|KEY| VALUE |
|--|--|
| DISCORD_TOKEN | `discord-bot-token` |




## Usage

developer mode : `npm run dev` ( nodemon required )
start : `npm start`
